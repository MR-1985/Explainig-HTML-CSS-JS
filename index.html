Übungen HTML und CSS


Dick und Kursiv



Hier steht ein Text in dicker Schrift und das Wort Schrift ist dick und kursiv.
Einen Text oder sogar nur einzelne Buchstaben, kann ich dick schreiben,
indem ich ein < b > vor und ein < /b > nach den Buchstaben, oder das Wort schreibe.

Kursiv schreibe ich, wenn ich ein < i > vor das Wort, oder den Buchstaben schreibe und dahinter dann < /i >

Zeilenumbruch



Dieser Text ist mit mehreren Einbru Umbrüchen versehen, die ich mit <br> eingeleitet habe.
So entsteht kein Absatz und man kann gut sehen, dass dieser Text hier zusammen gehört.
Merke: <br> benötigt kein schließenden Tag.

p-Tag



Dieser Text steht nach einem Absatz. Ein Absatz entsteht, wenn man den Text in einem < p > -Tag < /p > schreibt.

Ein < p > -Tag < /p > hilft uns also einen Text besser zu gestalten.

Durchgestrichen



Weiter oben ist ein Wort durchgestrichen dargestellt.
Das funktioniert, wie bei den anden veränderten Worten, nur das ich ein < s > vor und ein < /s > hinter das Wort, oder einen Buchstaben schreibe.

Platznutzung



Jeder < p > -Tag < /p > , sowie alle anderen Elemente, wie < div > (Container) , < h1-h6 > (Überschriften), < ul/ol > (ungeordnete Liste / geordnete Liste) , < table > (Tabelle)
nehmen im Code immer die volle Breite ein.
Hier ein Beispiel eines "div-Containers".

Warum stehen sie nicht nebeneinander? Sie hätten doch den Platz dazu!
Genau, weil sie die gesamte Breite des verfügbaren Platzes für sich reservieren.


Wenn man beide Container in anderen Übercontainern verschachtelt und diesem dann im CSS spezielle Eigenschaften, wie
display: flex;
justify-content: center;
flex-direction: column; ...usw.
zuteilt, dann gibt man ihnen Raum zum entfalten solange Platz genug da ist.

CSS ist wie eine neue Programmiersprache, nur das ich damit nichts anderes mache,
als meinen HTML-Code zu verschönern und die Website übersichtlich zu gestalten.

PS. die Buttons funktionieren nur, weil sie einen Code (Script) ausführen, der mit Java Script geschrieben wurde.
Java Script macht eine Website interaktiv.

"Inline-Elemente" wie "Links" < a > , oder "span" < span >, "Bilder" < img > , "Eingabefelder" < input > nur soviel Platz nehmen, wie sie brauchen.
